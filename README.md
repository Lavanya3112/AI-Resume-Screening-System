# AI Resume Screening and Candidate Ranking System

## Overview
This project implements an AI-inspired resume screening system that processes unstructured resume text, extracts key information, and ranks candidates based on skill alignment and experience.

## Features
- Resume parsing using rule-based NLP
- Skill and experience extraction
- Candidate-job matching
- Scoring and ranking system
- Explainable results

## Workflow
Load → Extract → Match → Score → Explain

## Technologies
- Python
- Regular Expressions (re)

## Sample Output
- Strong Candidate → Score: 100 | Level: Strong
- Average Candidate → Score: ~50 | Level: Average
- Weak Candidate → Score: 0 | Level: Weak
